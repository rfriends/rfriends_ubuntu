ubuntu/debian用のrfriends3インストールスクリプトです。

cd ~/  
wget -O rfriends_ubuntu-main.zip https://github.com/rfriends/rfriends_ubuntu/archive/master.zip  
unzip rfriends_ubuntu-main.zip  
sh rfriends3_ubuntu-main.sh
