ubuntu/debian用のrfriends3インストールスクリプトです。

cd ~/  
sudo apt install git  
git clone https://github.com/rfriends/rfriends_ubuntu.git  
cd rfriends_ubuntu  
sh rfriends3_ubuntu-main.sh
